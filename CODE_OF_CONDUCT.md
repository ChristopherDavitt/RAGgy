# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][cc].

By participating in this project — through issues, pull requests, discussions,
or any other contribution — you agree to uphold that standard. Please read the
linked document in full before contributing.

## Scope

This applies to all project spaces: GitHub issues, pull requests,
discussions, and any official channels operated by the project (chat rooms,
events, etc.). It also applies when an individual is representing the project
in public.
